# IoT Operations on Azure Local

Azure Local enables processing and managing IoT workloads at the edge, supporting real-time analytics and device management close to data sources.

## Overview
- Run IoT workloads and process data locally
- Supports real-time analytics and event processing
- Managed through Azure portal and APIs

## Supported Features
- Local device management
- Integration with Azure IoT Hub (hybrid)
- Disconnected and hybrid operation

## Deployment Notes
- Requires Azure Local infrastructure (Stack Edge, HCI, Hub)
- See Microsoft’s IoT on Azure Local deployment documentation

## Limitations
- Some cloud-only features may not be available
- Hardware and licensing requirements apply

> Data verified with Microsoft Azure Local documentation, February 2026.
