# VMs and Containers on Azure Local

Azure Local supports running traditional virtual machines (VMs) and Linux/Windows containers side-by-side, providing flexibility for diverse workloads.

## Overview
- Run VMs and containers on the same hardware
- Managed through Azure portal and APIs
- Supports hybrid and disconnected scenarios

## Supported Features
- Windows and Linux VM support
- Container orchestration with AKS
- Integration with local and hybrid storage

## Deployment Notes
- Requires Azure Local infrastructure (Stack HCI, Hub, Edge)
- See Microsoft’s documentation for VM and container deployment on Azure Local

## Limitations
- Some Azure cloud integrations may be limited
- Hardware and licensing requirements apply
