# Machine Learning and AI on Azure Local

Azure Local supports running machine learning (ML) and AI workloads at the edge, leveraging local compute and GPU resources for low-latency inference and analytics.

## Overview
- Run ML models and AI workloads locally
- Hardware acceleration with NVIDIA/Intel GPUs (where available)
- Managed through Azure portal and APIs

## Supported Features
- Local inference and analytics
- Integration with Azure ML for model management
- Disconnected and hybrid operation

## Deployment Notes
- Requires compatible Azure Local hardware (Stack Edge, HCI, Hub)
- See Microsoft’s ML/AI on Azure Local deployment documentation

## Limitations
- Training may require cloud resources
- Hardware and licensing requirements apply
