# Azure Services on Azure Local

This section details Azure services that run natively on Azure Local infrastructure (Azure Stack HCI, Hub, Edge, and related platforms). Only services that execute workloads locally are included.

## Included Services
- Azure Virtual Desktop (AVD) on Azure Local
- Azure Kubernetes Service (AKS) on Azure Local
- Azure App Services on Azure Local
- Azure SQL Managed Instance on Azure Local
- Machine Learning/AI at the Edge
- IoT Operations on Azure Local
- VMs and Containers
- Disconnected Operations

Refer to each sub-section for details, supported features, and deployment guidance.

> Data verified with Microsoft Azure Local, Stack HCI, Hub, and Edge documentation as of February 2026.
