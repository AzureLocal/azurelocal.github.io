# Azure SQL Managed Instance on Azure Local

Azure SQL Managed Instance can run on Azure Local, delivering managed database services with local data residency and compliance.

## Overview
- Managed SQL databases on-premises
- Supports hybrid and disconnected operations
- Managed through Azure portal and APIs

## Supported Features
- High availability and disaster recovery
- Automated backups and patching
- Integration with local and hybrid storage

## Deployment Notes
- Requires Azure Local infrastructure (Stack HCI/Hub/Edge)
- See Microsoft’s SQL Managed Instance on Azure Local deployment documentation

## Limitations
- Some cloud-only features may not be available
- Hardware and licensing requirements apply

> Data verified with Microsoft Azure Local documentation, February 2026.
