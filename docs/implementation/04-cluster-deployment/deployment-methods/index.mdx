## Method Comparison

| Aspect | Portal | ARM Template |
|--------|--------|--------------|
| **Ease of Use** | Visual wizard | Requires template knowledge |
| **Repeatability** | Manual | Fully automated |
| **Version Control** | No | Yes |
| **CI/CD Integration** | No | Yes |
| **Learning Curve** | Low | Medium |

## Azure Local Cloud Standard

:::tip Production Recommendation
For Azure Local Cloud Azure Local production deployments, use **Active Directory with ARM Template** for consistency and repeatability.
:::
