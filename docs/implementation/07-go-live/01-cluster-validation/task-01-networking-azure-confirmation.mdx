# Task 01: Networking & Azure Cloud Confirmation

## Overview

This step confirms that the **Client Implementation** and **Public Cloud Services** teams have completed their respective networking and Azure cloud component configurations and are ready for customer handover.

:::note
Technical validation procedures are performed in [Part 7: Cluster Testing & Validation](../../06-testing-validation/). This step confirms those validations are complete.
:::

---

## Networking/Firewall Confirmation

**Delivery Team**: Client Implementation

**Process Note**: Existing Azure Local Cloud process for Network and Security Turnover applies. Nothing net new for Azure Local specifically for customer turnover from a network or security perspective.

### Confirmation Checklist

| Component | Confirmed | Notes |
|-----------|-----------|-------|
| Firewall rules configured for Azure Local traffic | ☐ | |
| DHCP scopes configured for management network | ☐ | |
| DHCP scopes configured for customer networks | ☐ | |
| TOR switches configured for RDMA/RoCEv2 | ☐ | |
| Azure VPN Gateway connected | ☐ | |
| BGP addresses visible and routes propagating | ☐ | |
