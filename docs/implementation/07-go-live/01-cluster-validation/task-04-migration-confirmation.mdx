## Overview

This step confirms that the **Professional Services** team has completed all migration activities (if included in the project scope). If migration services are not part of this deployment, this step can be marked as N/A.

:::note
Migration is optional and depends on project scope. If no migrations are included, skip this section and proceed to the next step.
:::
