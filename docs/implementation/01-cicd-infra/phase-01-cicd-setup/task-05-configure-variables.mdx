## Objective

Configure CI/CD pipeline environment variables for each customer Azure subscription.

---

## Prerequisites

- [ ] Environments created (Step 04)
- [ ] Bootstrap values from Step 01 (ARM_CLIENT_ID, ARM_CLIENT_SECRET)

---

## Procedure

:::note
Follow these steps for **each** customer Azure subscription managed by Azure Local Cloud.
:::

1. Navigate to **Settings** → **CI/CD**
2. Expand **Variables**
3. Select **Add variable** for each required variable
