## Objective

Create a new source control project/repository for the deployment using the Azure Local template.

---

## Prerequisites

- [ ] Bootstrap complete (Task 01)
- [ ] Source control platform access (GitHub, GitLab, or Azure DevOps)

## Procedure

<Tabs groupId="scm-platform">
<TabItem value="github" label="GitHub">

### Create Repository from Template

1. Navigate to the [Azure Local organization on GitHub](https://github.com/AzureLocal)
2. Open the desired **template repository**
3. Click **Use this template** → **Create a new repository**
4. Configure:

| Field | Value |
|-------|-------|
| **Owner** | Your organization |
| **Repository name** | `<org>-azurelocal` (e.g. `iic-azurelocal`) |
| **Visibility** | Private |

5. Click **Create repository**
6. Wait for repository creation to complete

> **Tip:** If using GitHub Teams, move the repository to the appropriate team after creation via **Settings → Manage access**.

</TabItem>
<TabItem value="gitlab" label="GitLab">

### Create Project from Template

1. Navigate to your organization's group in GitLab
2. Click the **+** in the top-left corner → **New project/repository**
3. Select **Create from template**
4. Select the **Group** heading
5. Click **Use template** for the Azure Local template
6. Configure:

| Field | Value |
|-------|-------|
| **Project name** | `<org>-azurelocal` (e.g. `iic-azurelocal`) |
| **Project URL** | Your organization's group path |
| **Visibility** | Private |

7. Click **Create project**
8. Wait for project creation (may take a few minutes)

</TabItem>
<TabItem value="azdo" label="Azure DevOps">

### Create Project and Repository

1. Navigate to [Azure DevOps](https://dev.azure.com) and select your organization
2. Click **+ New project**
3. Configure:

| Field | Value |
|-------|-------|
| **Project name** | `<org>-azurelocal` (e.g. `iic-azurelocal`) |
| **Visibility** | Private |
| **Version control** | Git |
| **Work item process** | Agile (or your org standard) |

4. Click **Create**

### Initialize from Template

5. In the new project, go to **Repos → Files**
6. Click **Import a repository**
7. Enter the Azure Local template repository URL
8. Click **Import** and wait for completion

</TabItem>
</Tabs>
