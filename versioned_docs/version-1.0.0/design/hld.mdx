This document is under development. The High-Level Design will cover the overall architecture, network topology, and Azure management infrastructure for Azure Local deployments.

## Scope

The HLD will include:

- **Azure Management Architecture** — Management groups, subscriptions, and resource group hierarchy
- **Network Topology** — Hub-spoke or flat network design, VPN/ExpressRoute connectivity
- **On-Premises Architecture** — Physical rack layout, node configuration, and storage design
- **Identity & Security** — Entra ID integration, RBAC model, and conditional access policies
- **Monitoring & Operations** — Log Analytics, Azure Monitor, and alerting strategy
