## Overview

This stage configures the network infrastructure that is **already physically installed**. All hardware must be racked, powered, and cabled per [LLD specifications](../../../design/lld.mdx) before starting this stage.

:::warning Hardware Prerequisite
This stage assumes all network hardware is physically installed:
- OpenGear console server racked and powered
- Dell PowerSwitch TOR switches racked and powered
- All network cables installed per cabling design
- iDRAC interfaces connected to out-of-band management network
:::

## Prerequisites

- [ ] All network hardware physically installed and powered
- [ ] OpenGear console server accessible
- [ ] Dell PowerSwitch TOR switches accessible
- [ ] Firewall administrative access
- [ ] Network architecture documented
- [ ] VLAN assignments defined

---

## Validation Checklist

- [ ] OpenGear operational with all ports mapped
- [ ] iDRAC interfaces accessible via DHCP on OOB network
- [ ] Dell switches configured with QoS/DCB for RDMA
- [ ] VLT operational between TOR switches
- [ ] Storage VLANs 711-714 configured
- [ ] Firewall rules verified for Azure endpoints
- [ ] Microsoft Environment Checker validation passed
