# Phase 02: Monitoring & Observability

[![Azure](https://img.shields.io/badge/Platform-Azure_Local-0078D4?style=flat-square&logo=microsoftazure)](https://learn.microsoft.com/en-us/azure/azure-local/)

> **DOCUMENT CATEGORY**: Runbook   
> **SCOPE**: Monitoring and observability   
> **PURPOSE**: Configure Azure Monitor, Log Analytics, HCI Insights, and Update Manager   
> **MASTER REFERENCE**: [Microsoft Learn - Monitor Azure Local](https://learn.microsoft.com/en-us/azure/azure-local/manage/monitor-hci-single)

**Status**: Draft

This stage configures comprehensive monitoring and observability for Azure Local clusters and Arc-enabled servers. These configurations are applied post-deployment when Arc-enabled resources exist.

## Steps Overview

| Step | Title | Description | Dependencies |
|------|-------|-------------|--------------|
| 1 | [Configure Log Analytics Workspace](./task-01-configure-log-analytics-workspace.mdx) | Create and configure the Log Analytics workspace for all monitoring data | None |
| 2 | [Configure Azure Monitor Agent](./task-02-configure-azure-monitor-agent.mdx) | Deploy AMA to cluster nodes, configure Data Collection Rules | Step 1 |
| 3 | [Enable HCI Insights](./task-03-enable-hci-insights.mdx) | Enable Insights workbook for cluster health monitoring | Steps 1-2 |
| 4 | [Setup Alerting](./task-04-setup-alerting.mdx) | Configure alert rules and action groups | Steps 1-2 |
| 5 | [Deploy Windows Admin Center](./task-05-deploy-wac.mdx) | Deploy WAC on dedicated management server for cluster management | None |
| 6 | [Deploy OMIMSWAC Monitoring](./task-05-deploy-omimswac-monitoring.mdx) | Dell hardware monitoring via Windows Admin Center | Steps 1-2, 5 |
| 7 | [Configure Network Device Logging](./task-06-configure-network-device-logging.mdx) | Syslog forwarding from switches and firewalls to syslog server | Step 1 |

> **Note**: Azure Update Manager is covered in [Phase 20: Security & Governance](../phase-04-security-governance/task-05-configure-azure-update-manager.mdx).

## Prerequisites

- ✅ **Cluster Deployed** — Azure Local cluster registered with Azure Arc
- ✅ **Arc-Enabled Servers** — Cluster nodes registered as Arc-enabled servers
- ✅ **Log Analytics Workspace** — Created in Stage 06 (or create new in Step 1)
- ✅ **Syslog / NDM Server** — Deployed and accessible for network device logging (Step 7). See [Task 07](./task-06-configure-network-device-logging.mdx) for server requirements.
- ✅ **Windows Admin Center** — Deployed on dedicated WAC server (Step 5). See [Task 05](./task-05-deploy-wac.mdx) for server requirements.
