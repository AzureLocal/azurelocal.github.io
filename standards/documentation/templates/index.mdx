## How to Use

1. **Choose the appropriate template** based on your document type
2. **Copy the entire file** to your target location
3. **Replace all placeholder text** (text in `[brackets]` or `{braces}`)
4. **Update the frontmatter** with actual values
5. **Remove instruction comments** before publishing

## Template Types

| Template | Use For |
|----------|---------|
| [Index Template](./golden-index-template.mdx) | Category landing pages, product index pages |
| [Document Template](./golden-template.mdx) | Standard documents, runbooks, guides |
